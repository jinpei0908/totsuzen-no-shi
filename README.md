# totsuzen-no-shi

Totsuzen-no-shi (突然の死) ascii art generator plugin. It's a joke plugin.

## Install

Please install deno, then execute below command.

dein.vim

```
call dein#add('vim-denops/denops.vim')
call dein#add('jinpei0908/totsuzen-no-shi')
```
